# MnMn

MnMn は Windows 上で動作するWebサービス専用ブラウザです。

今のところ対応しているサービスは以下になります。

 * ニコニコ動画

# ダウンロード

 * インストーラー: [自動セットアップ](https://bitbucket.org/sk_0520/mnmn/downloads/MnMn-Setup.exe)
 * アーカイブ手動展開: [ダウンロード](https://bitbucket.org/sk_0520/mnmn/downloads/)を参照。
