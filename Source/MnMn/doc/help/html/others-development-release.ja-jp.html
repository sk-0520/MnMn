ヘルプ

前提条件

通常のビルドが出来たとしてもリリース用のビルドが出来るわけではありません。

ビルドには MSYS が必要になります。
もっと言うと Windows 用の git をインストールした際に同梱されている msys になります。

  • git が使用可能である必要があります。
  • ソースコード単品ではなくソースコード自体のリポジトリ内で作業する必要があります。
  • 全ての変更がコミット済みである必要があります。

リリースビルド

<MnMn>\build.bat を実行すると、 <MnMn>\Output\Release\ に ZIP アーカイブが出力されます。
アーカイブ名は app のバージョンになります。

実施される内容

  • 各バージョン情報の <YEAR> を実施年に置き換え。
  • appのバージョン情報に現在リポジトリのリビジョンを埋め込み。
  • 成果物から不要ファイルの除外。
  • 成果物のアーカイブ。
