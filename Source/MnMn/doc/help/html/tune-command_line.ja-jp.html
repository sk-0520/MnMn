ヘルプ

コマンドラインは /キー=値のペア形式か/キーのスイッチ形式で指定します。

キー 値 説明
log ディレクトリパス 指定ディレクトリにログファイルを出力します。出力はプログラム終了まで継続されます。
setting-root ディレクトリパス 指定ディレクトリをユーザーディレクトリとして起動します。
cache-root ディレクトリパス 指定ディレクトリをキャッシュディレクトリとして起動します。
execute-beta (スイッチ形式) β版実行を認識していることをappに伝えます。β版実行時にこのスイッチがない場合、起動時に確認ダイアログが表示されます。
smile-login-name ニコニコ動画ログイン名
smile-login-pass ニコニコ動画ログインパスワード
